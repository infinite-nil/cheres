# Mobile Engineer bar raiser

## Running on your device

```
$ git clone git@github.com:infinite-nil/cheres.git

$ ... cloning

$ cd cheres/

$ npm install

$ npx expo start
```

Based on your environment press `i` or `a` to open the desired simulator.
